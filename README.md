# Sentinel CRM

Personal & commercial insurance agency CRM for Sentinel Insurance, LLC.

## Features
- Client management (personal & commercial lines)
- Policy tracking with coverage types & carriers
- Sales pipeline & prospect management
- Email templates with mailto integration
- Activity logging & follow-up scheduling
- Calendar view & reporting dashboard
- CSV export

## Tech
Single-file React app. No build step required. Data stored in browser localStorage.

## Deploy
Hosted on Netlify. Auto-deploys on push to `main`.
